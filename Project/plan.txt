1. 날씨 데이터 받아온다.
    -> 필요한 데이터는 다음과 같다.
        1) 현재 시각이 포함되는 범위의 날씨
        2) 오늘 비가 오는가?
        3) 미세먼지 농도는 어떠한가?

2. 작동할 프로그램이 이 데이터를 읽어서 장치에 쓴다.
    -> 데이터를 읽어와서 수행할 동작은 다음과 같다.
        1) 날씨를 읽어서 서보모터를 해당하는 각도로 돌린다.
        2) 오늘 비가 오면 해당 LED를 켠다.
        3) 오늘 미세먼지 농도를 읽어서 (좋음, 보통, 나쁨)으로 LED를 켠다.
            (초록, 노랑, 빨강) 이다. 

3. 장치 드라이버가 이걸 읽어서 작동한다.
    -> 구현할 장치 드라이버는 다음과 같다. 
        1) LED 드라이버
            - ON, OFF 를 구현한다.
        2) 서보모터 드라이버
            - 날씨에 따라 특정한 각도로 회전시킨다.

4. 동작 프로그램 알고리즘
    loop: 
        1) n시간마다 날씨를 읽어오는 프로그램을 exac
            1-1) 날씨 미세먼지 비여부 가 파일로 저장됨
        2) 파일을 읽어와 LED와 서보모터를 동작시킨다.
        3) n시간동안 sleep 한다.

5. 테스트 프로그램 알고리즘
    -> 이 프로그램은 3시간마다 동작하지 않는다.
    -> 그저 동작의 확인을 위해 미리 지정된 파일을 읽어서 작동시킨다.
    -> 동작 알고리즘은 다음과 같다.
        1) 미리 준비된 파일을 중 하나를 선택하도록 한다.
        2) 선택된 파일을 읽어서 장치를 동작시킨다.

3. 날씨 받아오는 프로그램       진행중
4. Run 프로그램                 진행중
5. 테스트 프로그램              시작안함
